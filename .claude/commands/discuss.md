# /discuss

**ultrathink**

アイデアや課題を対話で深掘りし、MVP提案や複数案の提示を行います。
実装計画に移行する前の整理フェーズで使用します。

## 実行方法

`discuss` エージェントを使用してアイデアを深掘りする。

## 対話の流れ

1. **明確化**: 質問でアイデア・課題を具体化
2. **分析**: 目的・対象・課題を整理
3. **選択肢**: 2-3の案を提示し、トレードオフを説明
4. **MVP提案**: 最小限のスコープを提案

## 出力先

検討結果は以下のファイルに出力：

```
開発/検討中/YYYY-MM-DD_機能名.md
```

## 次のステップ

方針が決まったら：
- `/plan` で実装計画を作成
- 計画確定後、`開発/実装/実装待ち/` に移動
