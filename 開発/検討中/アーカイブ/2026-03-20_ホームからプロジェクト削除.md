# ホームからプロジェクト削除

## 概要

devtools ホーム画面からプロジェクトを削除できる機能を追加する。

## 削除対象

1. Docker コンテナ・ボリューム停止・削除（`docker compose down -v`）
2. プロジェクトディレクトリ削除（`rm -rf`）

Git リモートリポジトリの削除は含めない（危険すぎる）。

## UI設計

- 通常モード: プロジェクト一覧（カード表示）。削除ボタンは非表示
- 左下に「削除モード」ボタンを配置
- 削除モード: 各カードに赤い「削除」ボタンが出現
- 削除ボタン押下で確認ダイアログ「本当に削除しますか？」
- 「はい」で Docker 停止 + ディレクトリ削除を実行

iPhoneのホーム画面でアプリ削除するときと同じUX。

## バックエンド

- `DELETE /api/projects/:name` または `POST /api/projects/destroy`
  - `docker compose down -v` を実行
  - プロジェクトディレクトリを `rm -rf`
  - パストラバーサル防止必須

## 決定事項

- 確認は「本当に削除しますか？」の1回のみ（プロジェクト名入力は不要）
- Git リモート削除は含めない
