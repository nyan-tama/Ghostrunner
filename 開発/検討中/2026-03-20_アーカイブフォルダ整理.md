# アーカイブフォルダ整理

## 日付
2026-03-20

## 決定事項

### 1. Ghostrunner 自体のフォルダ整理

トップレベルの `開発/アーカイブ/` を廃止し、各サブフォルダにアーカイブを分散する。

**変更後の構成:**
```
開発/
  検討中/
    アーカイブ/     ← 既存 + 旧アーカイブの検討系ファイル移動
  実装/
    実装待ち/
    完了/
      アーカイブ/   ← 新設、旧アーカイブの plan/report ファイル移動
  資料/
    アーカイブ/     ← 新設
```

**ファイル仕分けルール:**
- `_plan.md` または `_report.md` → `開発/実装/完了/アーカイブ/`
- それ以外 → `開発/検討中/アーカイブ/`

### 2. /init での生成

新規プロジェクト作成時に `開発/` フォルダ群（アーカイブ含む）を生成するステップを追加する。
